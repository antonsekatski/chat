## Let's do this

![alt](https://media.giphy.com/media/S6qDFrTbdwSuQ/giphy.gif)

Install dependencies:

```sh
$ npm install
```

Start the local dev server:

```sh
$ npm run server
```

Open in browser:

```
http://localhost:8080
```
